## データ取り込み方針

### スプレッドシート → アプリへの反映方法

**推奨: ビルド時取得方式**

```
スプレッドシート変更 → npm run sync → JSON更新 → npm run build → デプロイ
```

- シラバスは年度初めに確定したらほぼ変わらない
- API Key不要、高速表示、オフライン可
